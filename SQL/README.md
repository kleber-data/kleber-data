# 📂 Meus Estudos de SQL

Este diretório contém uma coleção de repositórios, estudos e anotações relacionados ao aprendizado de bancos de dados relacionais e SQL.

## 📂 Estrutura da Pasta

* 📝 **[Anotações](./Anotações):** Anotações básicas sobre conceitos de banco de dados, comandos e sintaxes consolidadas.
* 🚀 **[Projetos](./Projetos):** Scripts SQL, modelagens de dados (DER/MER), consultas complexas e laboratórios práticos.
* 📜 **[Certificados](./Certificados):** Certificados de conclusão de cursos e módulos focados em bancos de dados e SQL.

## Objetivo

Facilitar a consulta rápida de comandos SQL, consolidação de boas práticas em bancos de dados relacionais e acompanhamento do aprendizado contínuo.
