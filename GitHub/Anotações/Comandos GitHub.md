## 🛠️ Comandos Rápidos
-## CONFIGURAÇÃO E INÍCIO

1. git init: Inicia um repositório Git na pasta atual.
    
2. git clone [url]: Baixa um projeto existente para sua máquina.
    
3. git config --global user.name "Seu Nome": Configura seu nome de usuário.
    
4. git config --global user.email "seu@email.com": Configura seu e-mail.
    

## CICLO DE ALTERAÇÕES (O "ARROZ COM FEIJÃO")

5. git status: Mostra o estado atual dos arquivos (modificados, novos, etc).
    
6. git add . : Adiciona todos os arquivos alterados para a "area de espera" (staging).
    
7. git add [arquivo]: Adiciona apenas um arquivo específico.
    
8. git commit -m "mensagem": Salva suas alterações com uma descrição do que foi feito.
    
9. git diff: Mostra exatamente o que foi alterado dentro dos arquivos antes do commit.
    

## SINCRONIZAÇÃO COM O SERVIDOR (GITHUB/LAB)

10. git pull: Puxa as novidades do servidor para o seu computador.
    
11. git push origin [branch]: Envia seus commits para o repositório remoto.
    
12. git remote -v: Lista os endereços remotos conectados ao seu projeto.
    

## BRANCHES E NAVEGAÇÃO

13. git branch: Lista as branches existentes.
    
14. git checkout [nome]: Muda para uma branch que já existe.
    
15. git checkout -b [nome]: Cria uma nova branch e já entra nela.
    
16. git merge [nome]: Une as alterações de outra branch na sua branch atual.
    

## HISTÓRICO E RECUPERAÇÃO

17. git log: Mostra a lista de commits anteriores.
    
18. git log --oneline: Mostra o histórico de forma resumida (uma linha por commit).
    
19. git reset --hard [hash]: Volta o projeto exatamente para o estado de um commit específico (cuidado: apaga alterações não salvas).
    
20. git stash: "Esconde" suas alterações temporariamente para você limpar sua pasta de trabalho sem precisar fazer um commit.
    
    
    # Estrutura de Títulos (Cabeçalhos)

`#` Título 1 (O maior) `##` Título 2 `###` Título 3 `####` Título 4 `#####` Título 5 `######` Título 6 (O menor)

## Estilos de Texto

- **Itálico:** Use um asterisco `*` ou underline `_` antes e depois da palavra.
    
- **Negrito:** Use dois asteriscos `**` ou dois underlines `__` antes e depois.
    
- **Negrito e Itálico:** Use três asteriscos `***` ou três underlines `___`.
    

## Listas

- Para criar listas, use o hífen `-` ou o asterisco `*` seguido de um **espaço**.
    
- Para sublistas, dê **dois espaços** antes do hífen.
